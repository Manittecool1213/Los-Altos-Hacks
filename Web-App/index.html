What is MLearn?

MLearn is at its core a writing tutor, hosted as a web-app. It uses Optical Character Recognition (OCR) on handwritten text to determine the legibility and readability of the text. Through this, MLearn is able to recognize text, and inform the user about his/her mistakes and inaccuracies. Through such 'handwriting tests', which can be no more than a sentence or two long, MLearn can produce a comprehensive summary, outlining the key letters or symbols which the user needs to work the most on.
